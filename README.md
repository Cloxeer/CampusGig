# GetCampusGig

**Small gigs. Big campus energy.**

Post a task. Pick one up. Earn rep — built for **NMSU** students.

---

Hey — I’m an NMSU student who helped build this. We took **2nd place** at a hackathon with it, and we’re keeping it going because campus runs on quick favors: coffee, notes, errands, the stuff you already slide into someone’s DMs for.

**GetCampusGig** is that idea, but clearer: you post a gig or pick one up, finish it, leave a review, and **Rep** actually means something here because sign-in is **@nmsu.edu** — classmates, not random internet.

## What you can do

- Browse and post gigs around campus.
- Request → get accepted → get it done → review each other.
- Watch your Rep and the leaderboard if you’re into that.

**Tech:** React · Supabase · Vercel — nothing exotic, just solid for shipping fast.

**Try it:** [getcampusgig.com](https://www.getcampusgig.com)
